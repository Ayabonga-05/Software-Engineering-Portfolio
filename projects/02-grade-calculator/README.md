# 02 — Grade Calculator

A command-line Python tool that takes a student's subject scores,
calculates their average, and prints a full report with letter grade.

## What it does

- Accepts any number of subjects and scores
- Calculates average and assigns a letter grade
- Identifies strongest and weakest subject
- Prints a clean formatted report

## Skills demonstrated

- Python functions
- Lists and list methods
- Input validation
- String formatting
- Loops
